Описание курса

Введение

Данный курс знакомит студентов с основами динамики полета. Каждый модуль предполагает практическую работу по реализации изученного материала в среде моделирования MATLAB.Simulink

Результат прохождения курса

Результатом курса является реализованная цифровая нелинейная модель динамики ЛА с контурами управления

Требования для прохождения курса

Для успешного прохождения курса студенты должны владеть материалом первого семестра курса теоретической механики

Оборудование для прохождения лабораторных работ

  • Компьютер (с операционной системой windows)

Программное обеспечение

  • Visual Studio Code
  • MATLAB.Simulink
